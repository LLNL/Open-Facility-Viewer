====================================
OFV - Open Facility Viewer
====================================


Keyboard/Mouse controls:

arrow keys - movement/direction
'u'/'n' - look up/down

LEFT mouse button - change look direction

'b' - reset position back to start
'm' - toggle motion model (between fly and FPS)
'SPACE' - toggle collision model
'-'/'=' - decrease/increase fly/walk speed
'['/']' - decrease/increase turn speed
